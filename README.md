💡 Electricity Bill Calculator (GUI Application)

A desktop-based GUI application developed to calculate electricity consumption based on appliance usage. The project demonstrates strong implementation of core Object-Oriented Programming principles.

🔹 Key Features

Appliance-based electricity usage calculation

Interactive and user-friendly GUI

Modular class-based architecture

OOP concepts: encapsulation, inheritance, abstraction, polymorphism

🛠 Technologies Used

Java, Java Swing, IntelliJ IDEA
